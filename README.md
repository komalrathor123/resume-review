# Resume Focus

A polished single-page resume review tool with a dark theme and modern typography.

## What it does

- Lets users paste a job description and resume text.
- Extracts keywords from the job description.
- Compares the resume to the job requirements.
- Provides a match score, matched and missing keywords, improvement suggestions, and a draft resume outline.
- Supports PDF upload for both the job description and resume.
- Includes user review cards and a simple review submission form.

## Files

- `index (1).html` - Main project file containing the app UI, styling, and logic.

## How to use

1. Open `index (1).html` in your browser.
2. Paste the job description and resume text.
3. Click **Review Resume**.
4. Use **Copy Draft** or **Download TXT** to capture the generated resume draft.
5. Optionally upload PDF files for the job description and resume.

## Notes

- The page now uses a dark theme and updated font styling.
- The app runs entirely in the browser with no backend required.
- For best performance, use modern browsers like Chrome, Edge, or Firefox.
